# Impoartant Directories

## scripts/
Under inside the admin accounts home directory is a scripts/ directory, which contains scripts for this server.

## /data
This directory contains downloads/ directory which contains the recently downloaded movies/shows.
Inside this data directory there is also a movies/ and a shows/ directory which contains all of the movies and shows
on this server. These two directories is where jellyfin finds media.

## /docker
This directory contains docker related files, for all the containers that is running on the server.
